This page allows searching through a corpus of many Rikchik utterances. This corpus is organized into files, which contain utterances, each of which has a number of fields and values. Only the "text" field is required.

The main input box takes a string that will be treated as a regular expression. By default this will be run against the starred fields in the table below in the corpus, and every matching entry will be returned.

The "optional field" input box takes a field or list of fields (space-separated). Only these fields will be searched.

Field name Field description
* text Rikchik-language text of utterance in ascii form
* loose A loose translation of the utterance into English
* literal A more literal translation of the utterance into English
* note A note about the utterance.
author The name of the human who produced this utterance.
source Where the utterance appeared - may include a URL if it appeared online.
date The date that the utterance appeared, in YYYY-MM-DD format.
color The canonicity of the utterance, if any. "green" indicates utterances that could theoretically have been uttered by a rikchik. "blue" indicates utterances about life on Earth, translated into Rikchik.
id The id of the utterance, which consists of a filename, a document number within that filename, and an utterance number within that filename, separated by colons.
